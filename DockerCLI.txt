Setting up MySQL container
- docker pull mysql
- docker run -p 3306:3306 --name mysql_dock --network new_bridge_network -h blahblah -e MYSQL_ROOT_PASSWORD=RootDock@SQL -d mysql

Setting up SpringBoot app container
- docker build -t bizexpense_image .
- docker run -p 8080:8080 --name bizexpense_container --network new_bridge_network -d -e HOSTNAME=blahblah bizexpense_image

Start/stop containers:
- docker container start/stop containername
